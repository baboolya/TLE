Проект по глубокому рефакторингу и доработке движка визуальной новеллы. Основная задача заключалась в разделении данных и логики, а также в оптимизации системы диалогов.

### Выполненные задачи:
• **Refactoring:** Переработка кода в структурированную систему на основе событий (Event-driven) и паттерна State Machine.
• **Narrative System:** Реализована гибкая система ветвления сюжета, позволяющая легко добавлять новые сценарии через внешние файлы/данные.
• **UI/UX Optimization:** Оптимизация отображения текстовых блоков и управления спрайтами персонажей.
• **Архитектурный подход:** Внедрение принципов SOLID для упрощения поддержки и расширения проекта.

### Технологический стек:
• C#, Unity, Unity Events, State Machine, UI Management.
